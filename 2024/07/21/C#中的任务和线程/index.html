C#中的任务和线程

1. Single thread

  • 单线程效果，线程卡顿

2. Multiple thread

  • 用委托或者lambda实现

3. Task

  • 用task来做任务管理，不需要先实例化thread

4. Synchronization

  • 同步执行方法

5. Await

  • 异步方法，方法前用async标记，后续任务需要等await标记任务执行完毕后再执行

6. WhenAll

  • 等到所有任务结束后再执行
